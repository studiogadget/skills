## デッキ構成パターン

### パターンA: 標準分析デッキ（6〜8枚）

最も汎用的な構成。データ分析・市場分析・戦略分析に適用。

| # | スライド種別 | 主な要素 | 必須/任意 |
|---|------------|---------|----------|
| 1 | タイトル | テーマ、サブタイトル、日付 | 必須 |
| 2 | エグゼクティブサマリー | KPIコールアウト＋発見事項3点 | 必須 |
| 3 | 状況/コンテキスト | トレンドチャート or 市場データテーブル | 必須 |
| 4 | 分析①（定量） | 棒グラフ/折れ線 ＋ テキスト注釈 | 必須 |
| 5 | 分析②（構造化） | 2x2マトリクス or 比較テーブル | 必須 |
| 6 | 分析③（深堀り） | マルチカラム ＋ 補足データ | 任意 |
| 7 | 示唆/提言 | 構造化テイクアウェイ＋アクション | 必須 |
| 8 | 付録 | 補足テーブル、方法論ノート | 任意 |

### パターンB: 提案書デッキ（8〜10枚）

クライアント向け提案、新規事業企画、投資提案に適用。

| # | スライド種別 | レイアウト |
|---|------------|----------|
| 1 | タイトル | ダークネイビー背景 |
| 2 | エグゼクティブサマリー | KPI＋要点3つ |
| 3 | 課題/背景 | チャート＋テキスト（2カラム） |
| 4 | 現状分析 | データテーブル or ウォーターフォール |
| 5 | 解決策/アプローチ | プロセスフロー＋詳細テキスト |
| 6 | 競合/市場分析 | 2x2マトリクス or 比較テーブル |
| 7 | 財務インパクト | ウォーターフォール or 積み上げ棒 |
| 8 | 実行計画 | タイムライン＋マイルストーン |
| 9 | リスク/対策 | テーブル（リスク/影響/対策の3列） |
| 10 | 次のステップ | 番号付きアクション＋担当者 |

### パターンC: 報告書デッキ（5〜6枚）

定期報告、進捗報告、月次レビューに適用。

| # | スライド種別 | レイアウト |
|---|------------|----------|
| 1 | タイトル | ダークネイビー背景 |
| 2 | サマリー＋KPI | KPIコールアウト4つ |
| 3 | 実績データ | チャート（棒/折れ線）＋テーブル併用 |
| 4 | 課題と対応 | Before/After比較 or テーブル |
| 5 | 示唆 | 構造化テイクアウェイ |
| 6 | 補足データ（任意） | 付録テーブル |

## スライドレイアウト選択マトリクス

伝えたい内容に応じて、適切なレイアウトを選択する判断基準。

### データの種類 → チャートタイプ

| 伝えたいこと | データの形 | 推奨チャート | 代替 |
|------------|----------|------------|------|
| カテゴリ間の大小比較 | カテゴリ×値 | 縦棒グラフ | 横棒グラフ（カテゴリ名が長い場合） |
| 時系列の推移 | 時間×値 | 折れ線グラフ | エリアチャート（累積を強調） |
| 構成比・内訳 | カテゴリ×割合 | 積み上げ棒グラフ | ドーナツチャート（5セグメント以下） |
| 増減の要因分解 | 要因×増減額 | ウォーターフォール | 積み上げ棒で代用 |
| 2軸での位置付け | X値×Y値 | 2x2マトリクス | バブルチャート |
| 項目の詳細比較 | 多項目×多属性 | データテーブル | ヒートマップ風テーブル |
| プロセス・手順 | ステップ列 | プロセスフロー | タイムライン |
| 段階的絞り込み | ステップ×数量 | ファネル | 横棒の段階減少 |
| Before/After | 2状態×項目 | 比較レイアウト | 2カラムテーブル |

### コンテンツの種類 → レイアウトパターン

| コンテンツ | 推奨レイアウト | ノート |
|----------|-------------|-------|
| KPI/主要指標の強調 | KPIコールアウト行 | 3〜4指標を大きな数値で横並び |
| データ＋解釈 | チャート左＋テキスト右 | 2カラム（6:4比率）|
| 複数の発見事項 | 番号付きリスト | 番号バッジ＋タイトル＋説明の3段構成 |
| 項目の体系的比較 | データテーブル | 交互行背景、ヘアラインボーダー |
| 組織/構造の図示 | カード型レイアウト | 3〜4カード横並び |
| 時系列イベント | タイムライン | 水平ベースライン＋マイルストーン |
| 結論＋根拠 | テイクアウェイ＋アクション | 上部に示唆、下部に推奨アクション |

## レイアウトパターン詳細

### 1. チャート＋テキスト（2カラム）

最頻出のレイアウト。左にビジュアル、右にテキスト解説を配置。

```
┌──────────────────────────────────────────────┐
│ アクションタイトル（洞察を述べる文）           │
│ ──────────────────────────────────────────── │
│                                              │
│  ┌──────────────┐  ┌───────────────────┐    │
│  │              │  │ セクションヘッダー  │    │
│  │   チャート    │  │ ─────────────     │    │
│  │  (棒/折れ線)  │  │ ● ポイント1       │    │
│  │              │  │ ● ポイント2       │    │
│  │              │  │ ● ポイント3       │    │
│  │              │  │ ● ポイント4       │    │
│  └──────────────┘  └───────────────────┘    │
│                                              │
│ 出所: xxx                          ページ N  │
└──────────────────────────────────────────────┘
```

- チャート: `x=0.5, y=1.2, w=5.0, h=3.2`
- テキストエリア: `x=5.8, y=1.2, w=3.7`
- テキスト各ポイントの左にバーティカルアクセント（幅0.06のロイヤルブルー矩形）

### 4. 2x2マトリクス

戦略的ポジショニング、優先度マッピングに使用。

```

│ ──────────────────────────────────────────── │
│                                              │
│       ┌──────────────┬──────────────┐        │
│   高  │   象限TL     │   象限TR     │        │
```

- 象限サイズ: 約3.5" × 1.5"

## コンテンツ組み合わせガイド

### 1スライドに入れる要素の判断基準

| 情報密度 | 推奨組み合わせ | 避けるべき組み合わせ |
|---------|-------------|-------------------|
| 高（分析スライド） | チャート×1 ＋ テキストポイント×3-4 | チャート×2以上 |
| 高（データスライド） | テーブル×1（全幅） | テーブル＋チャート |
| 中（サマリー） | KPIコールアウト ＋ 発見事項×3 | KPI×6以上 |
| 中（フレームワーク） | マトリクス×1 ＋ 補足テキスト | マトリクス＋テーブル |
| 低（タイトル/提言） | テキスト中心 ＋ 構造化リスト | テキストのみ |

### チャートの選択フローチャート

```
データがある？
├── Yes → 何を比較する？
│   ├── カテゴリ間の大小 → 棒グラフ
│   ├── 時系列の変化 → 折れ線
│   ├── 構成比 → 積み上げ棒 / ドーナツ
│   ├── 増減の要因 → ウォーターフォール
│   └── 相関/位置付け → 2x2マトリクス
└── No → 構造/概念を表現する？
    ├── 段階/手順 → プロセスフロー
    ├── 時系列 → タイムライン
    ├── 絞り込み → ファネル
    ├── 比較 → Before/After
    └── 分類/構造 → カード型
```
